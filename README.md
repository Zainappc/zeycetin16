MÜZİK WEB SİTESİ PROJESİ
Bu proje HTML,CSS ve JavaScript kullanılarak hazırlanmış bir müzik web sitesi çalışmasıdır.
PROJEDE KULLANILAN TEKNOLOJİLER
HTML
CSS
JavaScript
Proje İçeriği: Projede farklı müzik türlerine ait sayfalar ,görseller ve kullanıcı etkileşimi için JavaScript dosyaları bulunmaktadır.
AMAÇ:Bu çalışmanın temel amacı web geliştirme yapısını uygulamak, HTML sayfaları oluşturmak,CSS ile tasarım düzenlemek ve JavaScript 
kullanımı konusunda pratik yapmak amacıyla hazırlanmıştır.
